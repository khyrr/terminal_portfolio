## Running Locally

Clone the project

```bash
git clone https://github.com/0xkhyr/terminal_portfolio.git
```

Go to the project directory

```bash
cd terminal_portfolio
```

Remove remote origin

```bash
git remote remove origin
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
